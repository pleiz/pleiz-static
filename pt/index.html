Um novo olhar

"A Web é a humanidade conectada pela tecnologia." — Sir Tim Berners-Lee.

Este é um site. Ele é composto de palavras, as que você está lendo ou ouvindo agora mesmo. Você poderá fazer isso desde outro lugar, outro dispositivo, outro idioma, em um outro momento: do seu jeito. Mas você as está recebendo porque elas têm um espaço na Web.

Ter um espaço na Web significa ter uma voz. Ter uma voz significa visibilidade, oportunidades, presença, existir. Nós nos perguntamos o que implica ser uma pessoa digital. O que implica ter uma voz e um espaço. O que implica não tê-los.

Nós queremos dar um espaço igualitário a todas as pessoas, especialmente para aquelas que ainda não tiveram a oportunidade de expressar suas palavras. De existir digitalmente. De ter seus espaços.

Novos espaços

  • Seguros

  • Acessíveis

  • Inclusivos

  • Diversos

  • Autônomos

  • Customizáveis

  • Livres

  • Das pessoas e para todas as pessoas.

Essa é nossa visão como Pleiz: um olhar humano para os nossos espaços digitais. Um olhar humano para os espaços que já estão na Web, para aqueles que estão sendo criados e para aqueles que virão.

É hora de repensar e de construir, passo a passo, a humanidade conectada pela tecnologia: porque a Web é para todas as pessoas.

Vamos começar.
